# site_crawler

# 仕様
1. アクセスしたページの中のすべての画像を保存する
2. ページ中のすべてのURLを取得する
3. URL中のドメインまたはディレクトリ名が同じURLにアクセスする
4. 1.からアクセスしていないURLがなくなるまで繰り返す
